# 🚀 Guia Rápido: Hospedar no GitHub Pages

## Em 5 Passos

### 1️⃣ Criar Repositório
- Vá para [github.com/new](https://github.com/new)
- Nome: `rastreador-cxa`
- Clique em **"Create repository"**

### 2️⃣ Upload dos Arquivos
- Clique em **"Add file"** → **"Upload files"**
- Selecione: `index.html`, `README.md`, `.gitignore`
- Clique em **"Commit changes"**

### 3️⃣ Ativar GitHub Pages
- Vá em **Settings** (⚙️)
- Clique em **"Pages"** (esquerda)
- **Branch**: Selecione `main`
- **Folder**: `/` (root)
- Clique em **"Save"**

### 4️⃣ Aguardar Publicação
- Aguarde 1-2 minutos
- GitHub vai mostrar uma mensagem com o link

### 5️⃣ Acessar
```
https://SEU_USUARIO.github.io/rastreador-cxa
```

## ✅ Pronto!

Seu Rastreador CXA está online! 🎉

## 📌 Dicas

- **Usar Git (recomendado)**:
  ```bash
  git clone https://github.com/SEU_USUARIO/rastreador-cxa.git
  cd rastreador-cxa
  # Copie os arquivos aqui
  git add .
  git commit -m "Adicionar Rastreador CXA"
  git push
  ```

- **Personalizar domínio**:
  - Em Settings → Pages → Custom domain
  - Aponte seu domínio para o repositório

- **Atualizar arquivos**:
  - Faça push das alterações
  - GitHub Pages atualiza automaticamente (2-5 min)
